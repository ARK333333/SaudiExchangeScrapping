# SaudiExchangeScrapping
A Python program that utilizes Selenium and Pandas to scrap the following Saudi Exchange's fields from the main market page: 
- Ticker identifier  
- Last trade 
	- Price 
	- Trade change value
	- Trade change percentage 
- Cumulative
	- Number of trades 
	- Volume traded 
- Today's
	- High
	- Low
	- Open
